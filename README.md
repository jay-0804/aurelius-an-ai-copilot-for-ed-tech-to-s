# Aurelius: An AI Copilot for Ed Tech
=====================================

## Description
Aurelius is an innovative AI-powered copilot designed to support educational technology (Ed Tech) as a helpdesk. This project aims to provide an intelligent and automated solution to assist students, teachers, and administrators with their Ed Tech-related queries and issues. By leveraging the power of artificial intelligence, Aurelius seeks to enhance the overall learning experience and improve the efficiency of Ed Tech support services.

## Getting Started
---------------

To get started with Aurelius, follow these steps:

1. **Clone the repository**: Clone the Aurelius repository using the following command: `git clone https://github.com/jay-0804/aurelius-an-ai-copilot-for-ed-tech-to-s.git`
2. **Install dependencies**: Install the required dependencies by running `pip install -r requirements.txt`
3. **Configure the environment**: Configure the environment variables and settings as per the instructions in the `config` directory
4. **Run the application**: Run the Aurelius application using `python app.py`

## Tech Stack
-------------

Aurelius is built using the following technologies:

* **Frontend**: React.js
* **Backend**: Node.js with Express.js
* **AI Engine**: TensorFlow.js
* **Database**: MongoDB
* **Natural Language Processing (NLP)**: NLTK and spaCy
